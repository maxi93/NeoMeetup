## MANUAL FOR NOOBS

piccoli tips per import csv in arangodb

- tirare su un database (una volta acceduti in localhost:porta_arango, andare in voce -> database, crea nuovo --> <nome_db> )

- preprazione csv per import 
	
	esempio: member.csv/group.csv/relaztions-member-group

	cambiare nome colonna
	member_id -> _key
	group_id -> _key

	nelle relative relazioni bisogna aggioungere <nome_collection>/id per entrmabe le colonne
	row -> member/id,group/id
	inoltre cambiarte nome colonne in _from e _to

- import da shell windows:

1. dirigersi in pat/arango/../bin

2. IMPORT MEMBER --> arangoimp.exe C:\path\member_arango.csv  --type=csv --server.database test2 (il nome che avete scelto per il db sopra) --collection member --create-collection true

3. IMPORT GROUP --> arangoimp.exe C:\path\group_arango.csv  --type=csv --server.database test2 (il nome che avete scelto per il db sopra) --collection group --create-collection true

4. IMPORT REL --> arangoimp.exe C:\path\rel-mem-gro.csv  --type=csv --server.database test2 --collection member-of --create-collection true --create-collection-type edge

in webapp arango nella sezione graph:

new graph --> 	edge definitions: member-of
				fromCollections: member
				to<Collections: group
